Use Vercel for fast microVMs with free pause/resume and public HTTPS URLs.

## Set up

_`agentbox vercel login` (OIDC, token, or CLI sandbox login)._

## Prepare snapshot

_`agentbox prepare --provider vercel` bakes the base snapshot._

## Limits

_No nested containers, no SSH, max 4 ports, region `iad1`._

## Pause / resume

_Persistent snapshots auto-pause on stop and resume for free._
