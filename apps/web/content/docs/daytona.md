Use Daytona for managed sandboxes with fast snapshots and preview URLs.

## Set up

_`agentbox daytona login` for your API key and org._

## Prepare image

_`agentbox prepare --provider daytona` builds and registers the project image._

## Credentials & resync

_Shared per-org credential volumes and `agentbox daytona resync`._

## Preview URLs

_Signed CloudFront preview URLs for box web apps._
