## Shell

_`agentbox shell` for an interactive (tmux) shell, plus named sessions._

## VS Code / Cursor

_`agentbox code` to attach your editor via Dev Containers._

## Web app URL

_`agentbox url` to open the box's web app in your browser._

## Live screen

_`agentbox screen` to watch the box's desktop over noVNC._

## Open files

_`agentbox open` to browse the workspace in Finder._

## Dashboard

_`agentbox dashboard` — the interactive box list + session TUI._
